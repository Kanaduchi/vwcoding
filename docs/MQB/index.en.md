# MQB platform coding

!!! warning "Important information"
    When encoding cars manufactured in 2021 and older, the following conditions must be met to remove write protection in blocks:  
        - hood open  
        - handbrake is on

### Vehicle blocks and access codes

    > Block 01 – Engine – code: 27971

    > Block 02 – gearbox electronics  

    > Block 03 – ABS/ESP – codes depending on the options being changed: 20103, 40168, 11966, 25004, 40304 (for BDW – drying brake discs), 15081, 32304, 33776, 33777, 33799, 37201, 37202, 37203, 37483, 24990, 25327, 25377, 25757, 31857, 10815, 11122, 14108, 14451, 24435, 12345

    > Block 4B – Multifunctional module

    > Block 09 – On-board power supply control unit (BCM) - code: 31347  

    > Block 6D – Trunk lid electronics – code: adaptation - 12345, coding - 20103  

    > Block 13 – Auto-Distance / Adaptive Cruise Control – code: 20103, 14117  
  
    > Block B7 – Access and engine start authorization system

    > Block 17 – Instrument panel – code: 20103  

    > Block 18 – Autonomous heater – code: 80782  

    > Block 19 – Gateway  

    > Block 44 – Steering Assist – code: 19249  

    > Block A5 – assistant camera – code: 20103  

    > Block 5F – MMI/RNS – code: 20103  

    > Block 65 – Tire pressure – code: 20103  

    > Block 76 – Parking assistant

    > Block 82 – Projection on the windshield
