# Прошивки для камеры ассистентов

Для полноценной работы камеры ассистентов нужна правильная прошивка и параметрия к ней.  
Камеры могут быть 3 вариантов:  

* 5Q0... - MFK 1.0 (Lane Assist, Sign Assist, DLA)  
* 3Q0/3QD... - MFK 2.0 (Lane Assist, Sign Assist, Traffic Jam Assist, DLA)  
* 2Q0... - MFK 3.0 (Lane Assist, Sign Assist, Traffic Jam Assist, Pedestrian Assist, DLA)  

Параметрии для прошивок L и H с увеличенным временем предупреждения (60с) для разных машин можно найти на [https://mibsolution.one](https://mibsolution.one/#/1/9/Datasets/MQB/A5/60+%20Mod)

## MQB-Evo — камера A5 (5WA 980 653)

На платформе **MQB-Evo** используется камера **5WA 980 653** (не 3Q0/2Q0). Прошивки и параметрии **не совместимы** с классическим MQB без проверки.

| Аппаратная версия | Диапазон SW (пример) | Функции                                           | SFD       |
|-------------------|----------------------|---------------------------------------------------|-----------|
| 5WA980653A        | 3129 – 3303          | Lane Assist, Sign Assist, Travel Assist (базовый) | да        |
| 5WA980653D        | **3403+**            | + **Auto Lane Change** (с Side Assist, PLA)       | да        |
| 5WA980653         | 3405+                | TA 2024+, улучшенный TA (зависит от PR/рынка)     | да / SFD2 |

!!! warning ""
    - **Auto Lane Change** — см. [Travel Assist MQB-Evo](../MQB-Evo/travelAssist.md#auto-lane-change-opcionalno) и [доп. ассистенты](../MQB-Evo/driverAssistExtras.md).  
    - Параметрии для Evo выбирайте по **VIN / HW / SW** в ODIS или на [mibsolution.one](https://mibsolution.one) (раздел MEB/MQB Evo → A5).  
    - Flash камеры A5 на Evo только через **ODIS online** + SFD; ошибка параметрии = потеря ассистентов до восстановления backup.
