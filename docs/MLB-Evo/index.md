# Кодирование платформы MLB-Evo

!!! note "Важная информация"
    При кодировании MLB-Evo платформы необходимо выполнить следующие условия для снятия защиты от записи в блоки:  
        - открыт капот   
        - включено зажигание  
        - включен ручной тормоз
