# DICTIONARY OF GERMAN TERMS

* abschalten - turn off
* Anzahl - number, quantity
* Ausschaltgrenze - switch-off limit
* Bedienfolge - sequence of operations
* Begrenzungslicht - side light
* beidseitiges - one-sided
* Betaetigungen – starting, turning on, putting into action
* Bremslicht - brake light
* dauer - duration
* Dauerfahrlicht - DRL
* Defaultwert – default value
* Einschaltgrenze - switching limit
* Einschaltereignis - trigger event
* Fahrt - trip, ride
* Fahrertuerbedienung - driver's door controller
* Fensterheber - window lifter
* Freigabenachlauf - unlocking
* Freigabezeit - unlocking period
* Fernlichtassistent - high beam assistant
* Freischaltung - activation
* Frontwischer – front windshield wiper
* Funk - radio
* Fussraum – for legs
* grenze - limit
* Handbremse - handbrake
* Heck - rear part of the body
* Heckwischer – rear window wiper
* Helligkeitsschwelle - brightness threshold
* Hinweis - indicator
* Innenbeleuchtung – interior lighting
* Innenlichtinszenierung – atmospheric interior lighting
* kontrollleuchte - indicator
* Komfortbedienung - comfort functions
* Lichthupe - flash
* Lichtwarnungsverhalten - behavior of the warning about a forgotten light on
* Menuesteuerung - menu control
* Nachwischen – to wipe
* Nebelschlusslicht - rear fog lights
* Notlaufprogramm – emergency program
* Regenschliessen - closing in case of rain
* Richtung - direction
* Rueckmeldung - feedback
* Schalterstellung - position switch
* Scheinwerfer - headlights
* Scheinwerferreinigungsanlage (SRA) – headlight cleaning system
* Schlusslicht - rear light
* Spiegelanklappen - folding mirrors
* Spiegelabsenkung - lowering the mirror
* Standlicht - (dimensions) side lights
* Tag - day
* Tagfahrlicht - DRL
* Tagfahrlichtkontrollleuchte - DRL operation indicator
* Traenenwischen – adding drops
* Vorn - ahead
* Warngeschwindigkeit - warning speed
* Warnung - warning
* Werkseinstellung - factory setting
