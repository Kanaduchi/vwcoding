# Зарядка и HV-батарея

!!! warning ""
    Блоки **C5**, **8C**, **8B** (BMS) на MEB защищены **SFD** / **SFD2**. Лимиты DC/AC **задаёт BMS**, а не одна адаптация «максимальный ток».

## Что можно настроить штатно (без ODIS)

В меню **Зарядка** на ID.3/ID.4/Enyaq:

- **Снижение тока AC** (например 8 A вместо 16 A) — для слабой домашней сети  
- **Departure time** — отложенная зарядка по тарифу  
- **Минимальный SOC** (0–50%) — порог «не разряжать ниже»  
- **Оптимизация DC** — автопрогрев батареи по маршруту с зарядками (preconditioning)

Эти функции не требуют кодирования.

## Диагностика медленной зарядки (ODIS / OBD)

Полезные **измеряемые величины** (имена могут отличаться по SW):

| Блок | Что смотреть | Зачем |
|------|-------------|-------|
| **C5 / 8B** | Dynamic limit for charging (A) | Реальный лимит BMS сейчас |
| **C5 / 8B** | HV battery min/max temperature | Холодная батарея = низкий DC |
| **8E** | AC charge power / phases | Ограничение onboard charger |
| **19** | Charging mode | AC / DC / ошибка кабеля |

!!! tip ""
    Если **Dynamic limit** высокий, а станция даёт меньше — ограничение на стороне **зарядки**. Если лимит BMS низкий при тёплой батарее — ищите ошибки BMS, изоляции, SW-обновления.

## Ручной прогрев HV-батареи (8C, только диагностика)

!!! danger ""
    **Не штатная функция.** Используется энтузиастами перед DC-зарядкой в холод. Риск отказа гарантии, ошибок HV, некорректного завершения теста. Только на стоянке, с открытым доступом к ODIS.

На части ID.3/ID.4 (блок **8C** Battery Energy Module):

1. ODIS online, снять **SFD** на 8C.  
2. **Change Service → Development Mode** (иначе output test может не стартовать).  
3. **Output test** — HV battery heater (имя зависит от SW), ~300 с.  
4. Зажигание ON, капот можно приоткрыть при первом запуске.  
5. Контроль: мощность прогрева ~5–6 kW в live data; температура pack растёт.

Штатный **preconditioning по навигации** безопаснее и не требует development mode.

## Блок C5 — что обычно **не** кодируют

- Максимальная мощность DC (135 kW и т.д.) — заложена в BMS и температуре  
- Ёмкость батареи (58 / 77 / 82 kWh) — аппаратная, не «включается» кодом  
- Bidirectional charging (V2H) — зависит от PR, SW и рынка, не универсальная адаптация

При «пропаже» мощности зарядки сначала проверьте **ошибки C5/8C**, **12V** (слабый AUX → wakeups), **SW-обновления** через ODIS.
